How should your outstanding decisions list (ODL) be sorted?

Alphabetically

You will find your outstanding decisions list in the applicant data section of UCAS web-link.

What type of Star J (*J) return is needed?

Flat file

Star J (or *J) is a report that must be returned to the Higher Education Statistics Agency (HESA) at the end of each recruitment cycle. It contains personal information of each accepted applicant.

The return is available either as a flat file or as a UCAS link.
