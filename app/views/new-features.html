Here’s a list of new features we plan to introduce to Publish teacher training courses in the coming months. This list is only a guide and may change.

Recently released

You can now:

  • edit course vacancies
  • edit a location name and address
  • request a new course or location using a Google Form

May to June 2019

You’ll be able to:

  • change locations assigned to a course
  • add a new location
  • edit the UCAS Apply eligibility requirements for a course
  • edit the outcome of a course

July to September 2019

You’ll be able to:

  • rollover and prepare your courses for the next cycle
  • edit all aspects of a course
  • add a new course

Later

  • Improvements to locations to make courses easier to Find
