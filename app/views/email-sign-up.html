Publish teacher training courses - get started

Hello,

To start using the new service for managing and publishing teacher training courses, you’ll need to create a new account.

How to create a new account

  1. You’ll shortly receive an email from DfE Sign-in. This email will include a verification code, and a link to a page where you will need to enter this code and create a password for your new account.

  2. Once you’ve created your account, you’ll get an email from DfE Sign-in that tells you it’s been created successfully.

  3. You can then sign in to the new service.

Email becomingateacher@digital.education.gov.uk if you have any questions.
